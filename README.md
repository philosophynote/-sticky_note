# ノートアプリ

- 日々の生活の中で学んだことを記録するサイトです。
- 項目に従って記録していけば、アウトプットが楽になることを意図して作成しました。
- Google Book APIを活用したバーコードリーダーによって本を登録しやすくしています。

# 使用技術
- HTML5
- CSS3
- Bootstrap5
- Javascript
- Python 3.8.8
- Django 3.2
- SQLite3
- Google Books API

# 機能一覧
- ユーザ登録・ログイン機能
- CRUD機能
  - 音声入力機能
  - ISBNコード読み取りバーコードリーダー
- メモエディタ（quill）
